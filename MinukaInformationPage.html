Information Page of Student 4

- Accessibility

• The “alt” attribute is used in the < img > tag along with all the English names instead of using Sinhala (local) name when images are used ,so that a person with sight can recognize what the picture is using a screen reader.

• By clicking on the Text assigned for each item on the site map, you can procede to the respective page, else expand to see the sub content if it has any

• Default colors for background and text color is choosen in a way it is readable for everybody

- Typography

• Bold texts are used to emphasize usefull information and keywords with the descriptions of each image in the Gallery section

• Tag < abbr > is used to deliver more useful information to the user. Such text are underlined with a dotted line, hence it is easy to recognize amoung other words

- Colour (contrast) Background - White | Font - Black


- Colour (contrast) Background - Black | Font - White

• In the Galley page we have used a white background and black font as the basic view. It is also available to chose black background and white font which has passed the color contrast check done. Both these combinations are perfect and suitable for any person to use.

- Colour (balance / combination)

• Moreover these two setups can be used as day and night modes since durin daytime outside brightness is very high we need a suitable background out website, hence the white background. White font with Black background perfectly suits the night mod since the surrounding brightness is also low.

• Most of the tourists who visit Sri Lanka are determined to visit locations with high historical and religious values. Hence when using images for the gallery page I have used such places.

- Gestalt principles for grouping

* Continuation

It can be clearly seen in the site map page that user can continue to read though the full list by the way it is designed.

* Similarity

All the list items that fall under each level has a specific set of bullets used not by just increasing the Similarity but by having a advanced and easy to follow method.

* Closure

The ending of the Site map is clearly recognizable with the end of the white background and start of the page footer.
