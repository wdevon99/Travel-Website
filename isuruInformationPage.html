Information Page of Student 3

- Accessibility

• The colours selected are such that a person surffring from colour blindness wouldn't have a difficult identifing the diffrent colours.

• The respective labels for each radio button is linked to its labels so that the user can click the label which to activate the radio button, which will increase the accessibility.

• Red colour is used when marks are below 5 so that the people can identify that quiz marks are not acceptable.

- Typography

• Diffrent font sizes are used so user gets a idea about what are the headings, what are the sub headings, whats are the contents.

• Heading are bold so the it catches the eye and gives a idea when to start reading.

- Colour (contrast)

In the web site , mainly the text color is black and the background color is light yellow. The contrast check shows the results when the background is light yellow and the text is black. This helps user to easily indentify the font.

- Colour (balance / combination)

• Black text and light yellow background was used because, even in the gray scale it will appear the same, so a color blind person will not get affected when viewing the site.

• Diffrent background colour are used when the quiz is complete so the user gets a idea about his marks.

- Gestalt principles for grouping

* Similarity

Each questions and each answers looks the same.

* Proximity

The proximity between the questions and answers are almost the same so it can be identified easily.

* Closure

The ending of the quiz is clear defined by the submit button.

* Continuance

The quiz has a continuous down word flow, finally leading to the submit button
